## Flujo general recomendado para prototipado en Figma

1. Crear páginas de Figma por módulo: Estudiante, Docente, Administrador y General.

2. Importar primero las imágenes de `01_imagenes_ordenadas` como referencia visual limpia.

3. Importar las imágenes de `02_mockups_anotados` para exposición, explicación y revisión con docentes/estudiantes.

4. Usar los mapas de `03_mapas_de_flujo` como guía para conectar frames con flechas/prototipo interactivo.
